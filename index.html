3 componentes, 1 fluxo

01
Bridge Server
Rode node server.js na pasta bridge/. Ele cria uma API local em :3001 que conecta o site ao plugin.
02
Plugin Roblox
O plugin faz polling no bridge a cada 2s. Quando recebe um script, aplica automaticamente no Roblox Studio e abre o editor na linha certa.
03
Este Site
Gera o script via IA (OpenAI, Anthropic, Google, OpenRouter ou Groq) e envia ao bridge com POST /push-scripts.
SETUP RÁPIDO
1. Baixe a pasta bridge/ e instale Node.js (nodejs.org)
2. No terminal: cd bridge && node server.js
3. Instale EclipseCopilot_Plugin.lua no Roblox Studio (Plugins → Manage Plugins)
4. No plugin, clique em Conectar — o banner aqui ficará verde
5. Gere um script aqui e clique em Enviar para o Roblox Studio 🎉
